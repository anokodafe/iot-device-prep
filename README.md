# IoT Devices Pre-MSc Build

A 16-week roadmap building toward MSc Electronic & Computer Engineering, 
Major in IoT Devices, starting September 2026.

**Goal:** ESP32 fluency, real-time DSP, FreeRTOS, MQTT, and end-to-end IoT systems.

## Stack
- C / C++ (embedded)
- ESP32 with PlatformIO and ESP-IDF
- FreeRTOS, MQTT, Wi-Fi, BLE
- Python for data analysis
